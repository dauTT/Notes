# Remote

### Add remote repository

```text
# Add remote repository https://github.com/dauTT/indep_node_alarm
git remote add dauTT git@github.com:dauTT/indep_node_alarm.git
```

### List all remote repositories

```text
git remote -v
```

### Remove remote

```text
git remote rm remoteName

# example
git remote rm dauTT
```
